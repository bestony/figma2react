# figma2react

Export Figma Design into React Components

## Feature

- Export Figma Design into React Components

## Requirements 

- Node.js

## How To Use 

1. Install figma2react by run `npm install -g figma2react`
2. open a new folder
3. run `figma2react init`, then input your access tokenã€‚
4. run `figma2react export xxxxx`

## Others
### All command

#### `figma2react init`

Init Figma2React by Token

#### `figma2react export`

Export Figma Design

#### `figma2react whoami`

Get current user information.
